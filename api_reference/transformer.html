Transformer#

Base Transformer#

Transformer([model,interpret,algorithm])

An abstract base class for Transformers.

Transformer.fit(x,**params)

Fit this transformer to data

Transformer.data_transform(x)

Transforms data x to a new feature space.

Transformer.transform(x)

Wrapper for data_transform.

Transformer.fit_transform(x,**fit_params)

Fits this transformer to data and then transforms the same data

Transformer.inverse_transform(x_new)

Transforms data x_new from new feature space back into the original feature space.

Transformer.transform_explanation(explanation)

Transforms the explanation from the first feature space handled by this transformer to the second.

Feature Select Transformer#

FeatureSelectTransformer(columns,**kwargs)

A transformer that selects and re-orders features to match the model's inputs

FeatureSelectTransformer.fit(x,**params)

Saves the columns being dropped

FeatureSelectTransformer.data_transform(x)

Reorders and selects the features in x

FeatureSelectTransformer.transform(x)

Wrapper for data_transform.

FeatureSelectTransformer.fit_transform(x,...)

Fits this transformer to data and then transforms the same data

FeatureSelectTransformer.transform_explanation(...)

Transforms the explanation from the first feature space handled by this transformer to the second.
